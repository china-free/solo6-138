# Markdown Steganography Demo                       	 	  
	   			 	  		    				   		 	
This is a sample Markdown file to test the steganography tool.			  		 	  				 	    	  			 
 	 		  	    	   			 			  		 
## Features	 		 	 		   	 					 	   	 	 
					  	 	 				 					 				  
- Encode secret messages	  		 	 			  	  	 			   	 	 
- Decode hidden messages		 				  		 	  	 		 	    			
- Uses spaces (0) and tabs (1) at line endings			  		 	 		 	 		   	 					 
	   	 	 					  	 	 	  	    
## Usage 				    	  						   			 	 
  	  	  	       		   	  		 
Just run the tool with encode or decode subcommand.
